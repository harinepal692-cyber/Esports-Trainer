🎮 Esports Trainer Website

Esports Trainer is a free web-based platform designed to help gamers improve FPS skills like aim, movement, reaction time, and game sense.

🔥 Features:
- Aim training guides
- Movement improvement tips
- Reaction speed practice
- Game strategy for PUBG, Free Fire, Valorant
- Interactive training game

🌐 Built using HTML, CSS, and JavaScript.

🚀 Goal:
To help players become better esports gamers through simple and effective training.
